How It Works

A deep dive into the RePrompt pipeline — from raw pixels to reverse-engineered prompts.

01

The Pipeline

RePrompt follows a three-stage forensic analysis pipeline to reverse-engineer AI-generated image prompts.

I

Preprocessing

Your uploaded image is passed through OpenCV's preprocessing module. We extract foundational metadata — aspect ratio, mean brightness, global contrast, and dominant hue distributions using histogram analysis in HSV color space.

II

Physics Analysis

The physics engine applies computer vision techniques to classify lighting conditions, shadow hardness, depth-of-field characteristics, light direction, and contrast ratios. These measurements form the image's "physical DNA."

III

Gemini Vision Synthesis

The image and its physics metadata are sent to Google Gemini 2.0 Flash — a powerful cloud vision-language model. Gemini synthesizes a richly detailed generative prompt by combining what it sees with the verified physical measurements for maximum accuracy.

02

Extracted Metrics

Every image produces the following physical measurements.

Brightness Class
Categorizes lighting as low-key, balanced, or high-key based on mean luminance thresholds.
Depth of Field
Estimates shallow vs. deep focus using Laplacian variance sharpness scoring across image regions.
Shadow Hardness
Analyzes gradient transitions to determine if shadows are hard-edged (direct light) or soft (diffused).
Light Direction
Computes brightness distribution across quadrants to estimate the dominant light source direction.
Contrast Ratio
Measures the ratio between the brightest and darkest significant regions in the image.
Sharpness Score
Global Laplacian variance providing an absolute sharpness measurement across the full frame.
03

API Reference

RePrompt exposes two REST endpoints for integration.

POST /api/reprompt

Upload an image file to receive a reverse-engineered prompt and physics metadata.

Body multipart/form-data — field: file
Returns JSON — { reprompt, stats }
POST /api/improve

Send a basic text prompt to receive an enhanced, detail-rich version optimized for image generation.

Body JSON — { "text": "your prompt" }
Returns JSON — { result }

The Project

About

RePrompt is an open forensic tool for understanding AI-generated imagery at a physical level.

01

Why RePrompt?

AI image generators produce stunning results, but the prompts behind them are often lost, vague, or impossible to replicate. RePrompt bridges that gap.

By combining computer vision physics analysis with vision-language AI, RePrompt doesn't just describe what it sees — it understands how the image was lit, focused, and composed, then translates that understanding into a prompt you can actually use.

02

Tech Stack

⚡

FastAPI

High-performance async Python backend serving the analysis pipeline and static frontend.

👁

OpenCV

Computer vision engine powering the physics analyzer — brightness, shadows, DOF, and light direction extraction.

🧠

Gemini 2.0 Flash

Google's cloud vision-language model that synthesizes the final reverse-engineered prompt from image + metadata — no local downloads needed.

🎨

Vanilla Frontend

Pure HTML, CSS & JavaScript — zero frameworks, maximum performance, clean minimal aesthetic.
